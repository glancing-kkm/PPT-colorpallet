# Blueprint

## Overview

PPT 색상 팔레트를 설계하는 단일 페이지 웹앱이다. 사용자는 기준색을 HEX, RGB, 이미지 업로드로 지정하고, 용도를 체크한 뒤 추천 팔레트를 선택해 6장의 PPT 미리보기를 확인할 수 있다.

## Current Application Outline

- 4개의 행으로 화면을 구성한다.
- 1행은 기준색 설정 영역이다.
  - HEX 코드 입력
  - RGB 코드 입력
  - 이미지 업로드 기반 대표 색상 추출
  - 현재 기준색 프리뷰
- 2행은 용도 선택 영역이다.
  - 보고용
  - 제안용
  - 계획용
  - 분석용
  - 제출용
  - 각 항목은 체크박스를 텍스트 앞에 두고 hover 시 설명 툴팁을 표시한다.
- 3행은 추천 팔레트 영역이다.
  - 좌측 톤온톤 3종
  - 우측 톤인톤 3종
  - 각 팔레트는 10색, 1행 10열, HEX 코드 포함
  - 각 팔레트는 체크박스로 선택 가능
- 4행은 PPT 미리보기 영역이다.
  - 6개 슬라이드
  - 데스크톱 기준 2열 배치
  - 메인표지, 목차, 본문내용, 표, 차트, 그래프 구성
  - 선택한 용도와 선택한 팔레트에 맞춰 톤과 내용이 달라진다.
  - 마지막 하단에 선택 팔레트 클립보드 복사 버튼을 둔다.

## Visual And Interaction Notes

- 기본 화면은 스타터 템플릿에서 완전 교체한다.
- 시각 방향은 `Coolors` 트렌딩 팔레트 화면처럼 밝은 배경, 선명한 포인트 색, 긴 스와치 바, 짧은 라벨 중심으로 잡는다.
- 본문 글꼴은 현대적인 산세리프 계열을 사용하고, 라벨과 숫자는 더 강한 그로테스크 톤으로 구분한다.
- 추천 팔레트는 긴 수평 스와치 바 형태로 표현하고 HEX 코드를 셀 안에 직접 넣는다.
- 설명 문구는 가능한 한 한 줄 안에 들어가도록 짧게 유지하고, 필요한 경우 더 작은 보조 텍스트로 처리한다.
- 미리보기 슬라이드는 복잡한 장식보다 컬러 대비와 구조 인식이 빠른 카드형 레이아웃으로 유지한다.
- 최근 미리보기 리디자인은 더 세련된 프리미엄 프레젠테이션 톤을 목표로 하며, 영어 카피와 정제된 에디토리얼 구조를 사용한다.

## Current Change Plan

1. 기존 Hello World 화면을 4행 구조의 PPT 팔레트 설계 화면으로 교체한다.
2. 기준색 입력과 이미지 색상 추출 로직을 연결한다.
3. 용도 선택에 따라 성격이 반영되는 톤온톤 3종, 톤인톤 3종 팔레트 생성 로직을 만든다.
4. 선택 팔레트를 기반으로 6개의 PPT 미리보기를 렌더링한다.
5. 마지막에 선택 팔레트 복사 기능을 추가하고 기본 동작을 점검한다.

## Current Redesign Plan

1. 기존 올드한 톤의 히어로와 패널 스타일을 더 밝고 경쾌한 방향으로 교체한다.
2. 텍스트는 한 줄 우선 원칙으로 줄이고 길면 요약 표현으로 바꾼다.
3. 팔레트 카드는 `Coolors` 참고 방향의 긴 스와치 바 중심으로 재구성한다.
4. 슬라이드 미리보기의 카피와 폰트 크기를 줄여 카드 안에서 더 빠르게 읽히게 조정한다.

## Current Preview Refresh

1. PPT 미리보기 카피를 영어 기반으로 정리한다.
2. 차트와 그래프, 표 슬라이드를 더 조용하고 고급스러운 프리미엄 레이아웃으로 교체한다.
3. 슬라이드 카드의 색, 테두리, 타이포그래피를 더 정제된 방향으로 조정한다.
